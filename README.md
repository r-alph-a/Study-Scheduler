# Study Scheduler

A smart C++ application that helps students plan their study schedule by distributing study hours across subjects and chapters based on importance weights.

## How to run:

### Windows:

1. open cmd
2. git clone https://github.com/r-alph-a/Study-Scheduler.git
3. cd Study-Scheduler
4. Double click the run.bat file or run it in command prompt.

### MacOs / Linux:

1. open cmd
2. git clone https://github.com/r-alph-a/Study-Scheduler.git
3. cd Study-Scheduler
4. make
5. scheduler
