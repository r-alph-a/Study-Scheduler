# Study Scheduler

A smart C++ application that helps students plan their study schedule by distributing study hours across subjects and chapters based on importance weights.

## How to run:

### Windows:

git clone https://github.com/r-alph-a/Study-Scheduler.git
cd https://github.com/r-alph-a/Study-Scheduler.git
Double click the run.bat file or run it in command prompt.

### MacOs / Linux:

git clone https://github.com/r-alph-a/Study-Scheduler.git
cd https://github.com/r-alph-a/Study-Scheduler.git
make
scheduler
